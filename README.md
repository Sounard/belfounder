# befounders.eu — The Hackathon Playbook

The distilled operating system for running exceptional hackathons, live at
[befounders.eu](https://befounders.eu). Built from 35+ playbooks and research papers.

- `index.html` — landing + onboarding (path chooser, format picker, 15-rule checklist)
- `manual.html` — the Field Manual (8 chapters + cheat-numbers appendix)
- `timeline.html` — interactive Master Timeline (date-aware, two tracks, saved locally)
- `templates.html` — the swipe file (one-click copy)
- `agent-pack.html` — the Claude Code skill pack (install + skills)
- `downloads/` — `hackathon-agent-pack.zip`, `hackathon-playbook-full.zip`
- `assets/playbook.css` — design system (brand tokens: Stripe × Seven Events × Commons Hub)
- `hrmano/` — the previous hrmano landing page, preserved at [/hrmano/](https://befounders.eu/hrmano/)

Static site, no build step — GitHub Pages serves `main` at root.
